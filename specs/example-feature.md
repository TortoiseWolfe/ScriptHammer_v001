## Overview

**Feature**: User Dashboard
**Priority**: High

## Problem Statement

Users need a central place to view their activity and manage their account settings.

## User Stories

### US-001: View Dashboard
**As a** registered user
**I want to** see a summary of my recent activity
**So that** I can quickly understand my current status

**Acceptance Criteria:**
- [ ] Dashboard loads within 2 seconds
- [ ] Shows last 5 activities
- [ ] Displays current account status

### US-002: Quick Actions
**As a** registered user
**I want to** access common actions from the dashboard
**So that** I can complete tasks efficiently

**Acceptance Criteria:**
- [ ] Primary actions visible above the fold
- [ ] Actions reflect user's role/permissions
